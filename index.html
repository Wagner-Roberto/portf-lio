Messed: Jogo de Exploração Espacial

Messed é um jogo 2D desenvolvido em C++ com OpenGL. Nele, jogadores exploram sistemas subterrâneos contaminados e enfrentam monstros grotescos para escapar junto com seus amigos.

O foco do desenvolvimento está na movimentação suave, colisões, sistema de combate básico e iluminação dinâmica, usando bibliotecas gráficas e cálculos vetoriais.

Este projeto demonstra habilidades em lógica de jogos, gráficos com OpenGL e gerenciamento de recursos em C++.

Sabor na Rua: Site para Carrinho de Lanche

Sabor na Rua é um site criado com Next.js e TypeScript que simula um sistema de pedidos para um carrinho de lanche.

Ele conta com cardápio dinâmico, sistema de pedidos, mapa de localização e um design responsivo com foco em usabilidade. Animações e interatividade são garantidas com JavaScript moderno.

Este projeto destaca o uso de tecnologias modernas de front-end e boas práticas em design de interfaces.

KivDesk: Gestão Financeira Desktop

KivDesk é uma aplicação desktop criada em C# que permite ao usuário organizar suas finanças pessoais e empresariais.

Funcionalidades incluem controle de despesas e receitas, geração de relatórios, gráficos com Chart.js e integração com banco de dados SQLite. Possui ainda uma interface web opcional para visualização remota.

Este projeto demonstra integração entre desktop e web, manipulação de dados e foco em usabilidade.
